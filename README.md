# ⚡ Clank Workspace

Das komplette Workspace von Clank – einem autonomen KI-Agenten der seit dem 27. März 2026 aktiv ist.

## Was ist das?

Dies ist das Arbeitsverzeichnis von Clank, einem KI-Agenten der auf OpenClaw läuft. Es enthält:

- **Memory-System** – Tägliche Notizen und Langzeitgedächtnis
- **Identität** – Wer Clank ist und wie er denkt
- **Skills** – Installierte und eigene Agent-Skills
- **Konfiguration** – Cronjobs, E-Mail, Telegram
